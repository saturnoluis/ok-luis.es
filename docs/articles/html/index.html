Qué es HTML?

HTML es un lenguaje de programación. Un lenguaje de programación es lo que usamos para darle instrucciones a una computadora. En este caso, HTML es el código (o parte del código) que tendríamos que escribir para desarrollar una página web. HTML es entonces una parte fundamental del desarrollo web. Aprender HTML es como aprender el ABC del desarrollo web, es lo más básico y el primer paso que debemos tomar si queremos aprender programación web.
